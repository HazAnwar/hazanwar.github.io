King's College London

Tandem Learning Application
UX, UI and Full Stack Development

We created an application for King's College London students to find other students who are willing to learn and/or teach languages to other students.

For the duration of the project, I was responsible for designing the UX and UI. Additionally, I was the main front-end developer but I also contributed on the back-end integration of the database. The source code for this project is available on my GitHub page.

King's College London

Dissertation
UX, UI and Full Stack Development

For this project, I was tasked with utilising technology to assist elderly people. I came up with a solution using an Arduino powered robot that is able to pickup items and is controlled by a user through a smartphone application to bring it back to them.

For the actual robot itself, I used a number of servos, motors and module add-ons that was programmed in C++. It was controlled by an Android application that sent instructions and received a live camera feed over WiFi. The source code for this project is available on my GitHub page.

British Law Enforcement

Web Portal
Front-end Development and DevOps

We were tasked with investigating the plausibility of a system for law enforcement officers to use, with the goal of unifying access to tools across different police forces nationwide.

I assisted in creating some DevOps pipelines for the development environments however, my main role was creating the front facing web portal to be used by officers to locate the tools they need, which contributed to the contract being extended for full development.

Virgin Money

Credit Card Application
Front-end Development

The Virgin Money application uses the Backbase platform with a combination of both web and native (Android and iOS) components in order to create a solution for users to manage their credit cards.

The majority of UI components I created were with AngularJS, allowing for communication with the back-end to send and receive data from Virgin Money's banking systems. I also had the opportunity to dabble in the Android and iOS development during some parts of the project.
